AngularJS Basics

(Not for the dummies)

Introduction

AngularJS is a client-side JavaScript MVC (Model-View-Controller) Framework (like backbone.js and ember.js), which helps you to organize your JavaScripts in the MVC manner.

AngularJS is meant to replace jQuery, which is simply a JavaScript library. AngularJS is certainly more powerful than jQuery. However, it is much more difficult to learn and use, hence, not as popular as jQuery. jQuery also has a huge installed base with thousands of plug-ins to extend it functions.

AngularJS's features:

  • MVC Framework:
  • Single Page Application (SPA) Framework:
  • Facilitate unit testing and end-to-end testing.

This results in fewer lines, less boilerplate, cleaner, reusable and testable codes.

AngularJS 1.3, combined with jQuery and Bootstrap, lets you rapidly build webapp front-end that uses RESTful web services at the backend, for both desktops and mobiles.

Installation

AngularJS is a JavaScript file called angular.js. To use it, download angular.js or the minified angular.min.js from https://angularjs.org/ and include it in the HTML document. Alternatively, you can use a copy provided by a CDN (Context Distribution Network), e.g., check the Google Developers @ https://developers.google.com/speed/libraries/.

</body>

How It Works?
  1. Include the minified angular.min.js (or angular.js) under a <script>. In the above example, I used the Google's CDN, but you can also serve your local copy.
  2. You can place the <script> in the <head> section; or just before the end of body (</body>) for better responsiveness.

<!DOCTYPE html>
<!-- NGExDataBind.html -->
<html lang="en">
<head>
<meta charset="utf-8">
<title>AngularJS Example: Data Binding</title>
</head>
<body data-ng-app>
<input type="text" data-ng-model="yourName" placeholder="Enter your name">
<p>Hello, <span data-ng-bind="yourName"></span></p>
<p>Hello again, {{yourName}}</p>
<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.3.15/angular.min.js"></script>
</body>
</html>

Load the HTML file on a web browser (e.g., Chrome, Firefox). Enter something on the input text field, and observe the results.

How It Works?
  1. We bind the <body> element as an AngularJS app via the HTML attribute data-ng-app. You can also use attribute ng-app without the data-. But, ng-app will fail HTML5 validation, as HTML5 does not define such an attribute. On the other hand, HTML5 defines data-* as so-called custom attributes, which will not be validated. AngularJS typically binds to HTML DOM element via data-ng-* (or ng-*) attributes.
  2. This example illustrates data binding between the value of an input field and a DOM element. We named the input field via an HTML attribute data-ng-model (or ng-model). After that, we can bind the input value to another DOM element via data-ng-bind (or ng-bind). Once bound, whenever the input value changes, AngularJS updates the bound element and refreshes the UI.
  3. AngularJS further simplifies the binding syntax to {{var-name}}, as it is used frequently.
