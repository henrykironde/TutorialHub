How to Setup and Access NTU VPN

 

Setting up NTU VPN Client for Windows 8 and Windows 10

  1. Search ⇒ vpn
  2. Choose "Manage Virtual Private Networks (VPN)" or "Change VPN Settings".
  3. Add a VPN:
    • Provider: Microsoft (for Windows 10: Windows (Built-in))
    • Connection Name: NTU VPN (or your perferred name)
    • Server Name or Address: vpngate.ntu.edu.sg
    • VPN Type: Automatic
    • Username: student\yourUsername (NOTE: use backslash)
    • password: xxxxxxxx
    • Save
  4. To connect to the network via VPN: On the taskbar, click "Network" icon ⇒ Connect to the VPN created.

Setting up NTU VPN Client for Windows 7

  1. Control Panel ⇒ Network and Sharing Center
  2. Set up a new connection or network ⇒ Connect to a workplace ⇒ No, Create a new connection ⇒ next
  3. Use my Internet connection (VPN)
    • Internet address: vpngate.ntu.edu.sg ⇒ next
    • User name: student\yourUsername (NOTE: use backslash) ⇒ connect
  4. To connect to the network via VPN: On the taskbar, click "Network" icon ⇒ Connect to the VPN created.

Setting up NTU VPN Client for Ubuntu

  1. System Settings ⇒ Network ⇒ VPN Connections ⇒ Configure VPN ⇒ Add ⇒ VPN PPTP ⇒ Create
  2. Enter:
    • Connection name: NTU VPN
    • Gateway: vpngate.ntu.edu.sg
    • Username: student/yourUsername
    • Advanced: select Point-to-point encryption (MPPE)

Setting up NTU VPN Client for Mac OS X 10.7 to 10.9

  1. Click "Apple" icon ⇒ System Preferences ⇒ Network ⇒ "+"
  2. Enter:
    • Interface: Select "VPN"
    • VPN Type: PPTP
    • Service Name: NTU VPN ⇒ Create
  3. Configure:
    • Server Address: vpngate.ntu.edu.sg
    • Account Name: student\yourUsername (NOTE: Use back-slash)

If you encounter "DNS error", "DNS loopup failed", or "Can't find the server", check the "Proxy" (under "Advanced Settings") and select "Automatic Discovery". You need to refresh you page, or even disconnect and re-connect the VPN, after changing these settings.

See http://www.ntu.edu.sg/cits/gettinghelp/getmeconnected/Pages/configuringMacOS.aspx for screen shots.
